## Getting started

For to run the project locally:

1. Clone the repository:
   git clone <repo. name>

Navigate into the project folder:
cd <repository-folder>

2. Running the Server
   Open your terminal and navigate to the server folder:
   cd server

Start the server:
node server.js

3. Running the Frontend
   Navigate to the frontend folder in your project directory
   -- at terminal open cd client
   -- install npm
   and start frontend with: npm run dev
